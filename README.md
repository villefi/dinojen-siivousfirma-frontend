Beginning for Cleaning company project.

Technologies: React.js, Redux, axios



## Package installation
```bash
$ npm install
```

## Use development server

```bash
$ npm start
```

## Build assets
To put compiled files into `static` directory, type the following command.

```bash
$ npm run build
```

Set env

```bash
$ source .env
```
